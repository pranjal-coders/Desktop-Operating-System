2. Double-click desktop icons to open applications
3. Drag windows by their title bar to reposition

## Applications

### Notepad
- Type text in the editor
- Click "Clear" to erase all text
- Click "Download" to save as a `.txt` file

### Calculator
- Click number buttons to enter values
- Use operator buttons (+, -, *, /, %) for operations
- Press "=" to calculate result
- Press "C" to clear

### Browser
- Enter URLs in the address bar and click "Go"
- Use quick shortcuts for Wikipedia, ProxySite, and GitHub
- Navigate with back (‚Üê), forward (‚Üí), and reload (‚ü≥) buttons
